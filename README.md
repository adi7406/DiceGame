# 🎲 Dice Game

A simple web-based Dice Game built using HTML, CSS, and JavaScript.  
Each time the page is refreshed, two dice are rolled randomly and the winner is displayed.

---

## 🚀 Live Demo

👉 https://adi7406.github.io/DiceGame/

---

## 🧠 How It Works

- Two random numbers (1–6) are generated using JavaScript
- Each number corresponds to a dice image
- The images are updated dynamically
- The winner is decided based on the higher number

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🐛 Common Issues

- **Images not loading on GitHub Pages**
  - Ensure file names are lowercase
  - Avoid leading `/` in image paths

- **JavaScript not working**
  - Check if `script.js` is correctly linked
  - Verify file exists in the repository
